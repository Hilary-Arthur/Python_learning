## for循环
+ for i in 字符串，即从字符串中每次取一个出来处理
+ range语句
  + 语法1：range(num)获取一个从0开始到num结束的数字序列,不包含num自身
  + 语法2：range(num1,num2)从num1开始，到num2结束的数字序列不包含num2
  + 语法3：range(num1,num2,step)从num1开始，到num2结束，数字之间的步长（间距为step）的数字序列不包含num2
