## 标识符
+ 指用户在编程的时候所使用的一系列名字，用于给变量、类、方法等命名
+ 命名规则：
    + 内容限定（英文、中文【不推荐】、下划线、数字【不能使用在开头】）
    + 大小写敏感
    + 不可使用关键字
+ 标识符的命名规范
    + 见名知意
    + 简洁
    + 多个单词组合时，使用下划线分隔多个单词

## 字符串
+ 定义形式:单引号定义法（''）、双引号定义法（""）、三引号定义法（""""""）
+ 多重引号
+ 字符串拼接（字面量之间拼接、字面量和变量之间拼接）
### 字符串格式化方式一（实现对字符串和变量的快速拼接）
+ 通过占位符来拼接%s转化为字符串、%d转化为整数、%f转化为浮点数类型
+ 格式化的精度控制
  + 使用辅助符号m.n来控制数据的宽度和精度
  + m控制宽度，要求是数字，设置宽度小于自身不生效,大于用空格补上
  + .n控制小数点精度，要求是数字，会进行四舍五入
### 字符串格式化方式二
+ 在字符串前添加f（意思为format），语法为：
  ```
  f"内容{变量}"
  ```
### 对表达式进行格式化
+ 表达式：一条具有明确执行结果的代码语句，例如：1+1 or 5*2
