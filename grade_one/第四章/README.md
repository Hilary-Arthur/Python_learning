# 函数
+ 已组织好的/可重复使用的/针对特定功能
## 函数的定义语法
```
def 函数名(传入参数):
    函数体
    return 返回值
```
+ 定义函数后，还需要调用函数
+ 应当先定义函数，后调用函数
+ 注意:
  + 参数不需要可以省略
  + 返回值不需要可以省略
## 函数的传入参数
+ 在函数进行计算的时候，接受外部（调用时）提供的数据
## 函数的返回值
+ 函数返回值的定义
  + 函数体在return后就已经结束了
+ None类型
  + 如果函数没有return语句，那么函数的返回值为None
  + 因此无论函数有没有返回值，都存在一个return语句，当函数没有返回值时默认返回一个None
