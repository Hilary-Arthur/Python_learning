# 函数的进阶
## 函数的多返回值
+ 在return中返回多个值，按照返回值的顺序，写对应顺序多个变量接收即可
+ 变量之间使用逗号隔开
## 函数多种传参方式
+ 常见的4种形式：位置参数、关键字参数、缺省参数、不定长参数
+ 位置参数：
  + 调用函数时根据函数定义的参数位置来传递参数
  + 传递的参数和定义的参数顺序及个数必须一致
  + ```angular2html
    def user_info(name,age,gender):
        函数内容
    user_info('Tom',20,'男')
    ```
+ 关键字参数:
  + 调用函数时通过'键=值'的形式传递参数
  + 作用：让函数更加清晰，容易使用，同时消除参数的顺序需求
  + 关键字传参可以和位置参数混用，但是必须位置参数在前，且匹配参数顺序
+ 缺省参数：
  + 也叫默认参数，用于定义函数，为参数提供默认值，调用函数时可不传该默认参数的值
  + 注意：所有位置参数必须出现在默认参数前，包括定义和调用，默认值可以传参修改
  + 注意：默认值的设置必须在最后，而不能放在传入的非默认参数的前面
+ 不定长参数（可变参数）：
  + 用于不确定调用的时候会传递多少个参数的场景
  + 作用：当调用函数时不确定参数个数的时候，可以使用不定长参数
  + 不定长参数类型：
    + 位置传递:
      + 所有传进的参数都会被args收集，会根据传进参数的位置合并为一个元组，args是元组类型
      + ```angular2html
          def user_info(*args):
              print(args)
        ```
    + 关键字传递：
      + 参数是"键=值"形式的形式的情况下，所有的"键=值"都会被kwargs接收，同时会组成字典
      + ```angular2html
          def user_info(**args):
              print(args)
        ```
## 匿名函数
+ 函数作为参数传递，即函数传入的参数也可以是函数
+ lambda匿名函数
  + 没有函数名称，只可临时使用一次
  + 语法:lambda 传入参数:函数体(关键字)
