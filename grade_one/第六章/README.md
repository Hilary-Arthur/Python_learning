# 函数的进阶
## 函数的多返回值
+ 在return中返回多个值，按照返回值的顺序，写对应顺序多个变量接收即可
+ 变量之间使用逗号隔开
## 函数多种传参方式
+ 常见的4种形式：位置参数、关键字参数、缺省参数、不定长参数
+ 位置参数：
  + 调用函数时根据函数定义的参数位置来传递参数
  + 传递的参数和定义的参数顺序及个数必须一致
  + ```angular2html
    def user_info(name,age,gender):
        函数内容
    user_info('Tom',20,'男')
    ```
+ 关键字参数:
  + 调用函数时通过'键=值'的形式传递参数
  + 作用：让函数更加清晰，容易使用，同时消除参数的顺序需求
  + 关键字传参可以和位置参数混用，但是必须位置参数在前，且匹配参数顺序
