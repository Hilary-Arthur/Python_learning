# 文件操作
## 文件的编码
+ 编码技术：翻译的规则，记录了如何将内容翻译成二进制，以及如何将二进制翻译回可识别内容
+ 常见编码:UTF-8(全球通用的编码格式),GBK,Big5
## 文件的读取
+ 文件类型:文本文件、视频文件、音频文件、图像文件、可执行文件等多种类别
+ 常见的文件操作：打开、关闭、读、写
+ 操作步骤：
  + 打开文件
    + 语法：open(name文件名可以包含具体路径,mode打开文件的模式,encoding编码格式)
    + f = open('python.txt','r'encoding='UTF-8'不能使用位置参数)
    + 常见的模式:'w'、'r'、'a'(同c语言)
  + 读写文件
    + 文件的读取:
      + read()方法:文件对象.read(num)，读取定长
      + redlines()方法：按照行的方式将文件中的内容一次性读取，返回的是一个列表
      + readline()方法：一次读取一行内容
      + for循环读取文件行
  + 关闭文件:close()方法
  + 注意：可以只打开和关闭而不读写
  + with open语法：
    + ```angular2html
      with open("python.txt","r")as f:
        f.readlines()
      ```
    + 可以在操作完成后自动关闭文件，避免遗忘close方法
## 文件的写入（w）
+ ```
  f.write("hello world")
  f.flush()内容刷新
  ```
+ close方法中内置了flush的功能
+ w是从头开始复写全部内容
## 文件的追加
+ 与w模式不同的是，a是在文件后面进行追加
