# 数据容器
一种可以容纳多份数据的数据类型，容纳的每一份数据称之1个元素
每一个元素，可以是任意类型的数据，如字符串、数字、布尔等

## 数据容器的类型
+ 列表（list）、元组（tuple）、字符串（str）、集合（set）、字典（disc）

## 列表
### 字面量
```angular2html
    [元素1,元素2,元素3,...]
```
### 定义变量
```angular2html
    变量名称 = [元素1,元素2,元素3,...]
```
### 定义空列表
```angular2html
    变量名称 = []
    变量名称 = list()
```
+ 列表可以一次存储多个数据，且数据的类型可以不同,支持嵌套
+ 使用列表的下标索引从列表中取出元素(可以反向索引，从后向前)

## 列表的常用操作（列表的方法）
+ 插入元素
+ 删除元素
+ 清空元素
+ 修改元素
+ 统计元素数
### 方法
+ 在python中，如果将函数定义为为class类的成员，那么函数会称之为方法
+ 函数的使用和方法的使用不同:
```
函数的使用:num = add(1,2)
方法的使用:student = Student()
          num = student.add(1,2)
```
### 列表的查询方法
+ 查找指定元素在列表的下标，如果找不到就报错ValueError
+ 语法：列表.index(元素)
### 列表的修改方法
+ 列表[下标] = 值
### 列表的插入方法
+ 语法:列表.insert(下标,元素)，在指定的下标位置，插入指定元素
### 列表的追加元素
+ 列表append(元素)，将指定元素，追加到列表的尾部
+ 列表.extend(其他数据容器)，将其他数据的内容取出，依次追加到列表尾部
### 列表元素的删除
+ 语法1:del列表[下标]
+ 语法2:列表.pop[下标]
+ 删除某元素在列表中的第一个匹配项
  + 语法:列表.remove(元素)
### 列表清空
+ 列表.clear()
### 统计列表元素的数量
+ 语法：列表.count(元素)，放入需要统计的元素
### 统计列表中全部元素的数量
+ 语法:len(列表)

## 列表的遍历、迭代
+ while循环
  + 通过下标索引的方式，将每次的下标对应的元素依次取出来
+ for循环
  + 通过i取出arr中的数据

## 元组tuple
+  列表可以修改，而元组一旦定义完成不可以修改
+  定义方法:
  + 定义元组字面量(元素,元素,元素,...,元素)
  + 定义元组变量 变量名称 = (元素,元素,元素,...,元素)
  + 定义空元组
    + 变量名称 = ()
    + 变量名称 = tuple()
+ 注意事项：单个元素的元组需要再该元素后面添加一个逗号，否则会被识别为其他数据类型

## 元组的相关操作
+ index()、count()、len()
  + index()为查找某个元素的下标
  + count()为统计某个数据在元组中出现的次数
  + len()为获取元组长度

## 字符串str
+ 以数据容器的视角重新认识字符串
+ 字符串可以存放任意数量的字符

## 字符串的相关操作
+ 通过下标索引取值
+ index方法
+ replace方法:字符串的替换
  + 语法:字符串.replace(字符串1,字符串2)
  + 功能:将字符串内的全部:字符串1,替换为字符串2
  + 不是修改字符串本身，而是得到了一个新字符串
+ 字符串的分隔:
  + 语法:字符串.split(分隔符字符串)
  + 功能：按照指定的分隔符字符串，将字符串划分为多个字符串，并存入列表对象中
  + 注意：字符串本身不变，而是得到一个列表对象
+ 字符串的规整操作
  + (前后去空格)
    + 语法:字符串.strip()
  + 去前后指定字符串
    + 语法:字符串.strip(字符串)
+ 统计字符串中某个字符串出现的次数:count
+ 统计字符串的长度:len()

## 序列
+ 序列是指：内容连接、有序，可使用下标索引的一类数据容器
+ 列表、元组、字符串均可以视作序列

## 序列的操作
+ 切片操作:从一个序列中取出一个子序列
  + 语法:序列[起始下标:结束下标:步长]
  + 注意：此操作不会影响序列本身，而是得到一个新的序列

## 集合
+ 列表、元组、字符串均支持重复元素且有序
+ 若需要进行去重处理，则使用集合，集合使用{}表示
+ 集合的定义:
  + {元素,元素,元素,...,元素}
  + 变量名称 = {元素,元素,元素,...,元素}
  + 定义空集合 = set()
+ 集合是无序的，所以集合不支持下标索引访问

## 集合的常用操作
+ 添加新元素.add
+ 移除元素.remove(),移除指定元素
+ 随机取出元素
+ 集合的清空
+ 集合的差集集合1.difference(集合2)，集合1有而集合2没有的
+ 消除两个集合的差集
  + 语法:集合1.difference_update(集合2)
  + 功能:对比集合1和集合2，在集合1内，删除和集合2相同的元素
  + 结果：集合1被修改，集合2不变
+ 集合的合并:集合1.union(集合2)
  + 功能:将集合1和集合2合并为一个新的集合，原集合不变
+ 集合的遍历:
  + 集合不支持下标索引，因此没有办法while()
  + 但是可以使用for循环遍历
